<div class="markdown-heading"><h1 class="heading-element">react-calendar</h1><a id="user-content-react-calendar" class="anchor" aria-label="Permalink: react-calendar" href="#react-calendar"><span aria-hidden="true" class="octicon octicon-link"></span></a></div>
<div align="center">
  <a target="_blank" rel="noopener noreferrer nofollow" href="https://camo.githubusercontent.com/52917c4e3af39a7ca8c3b140dcd4aab5f51aae98a8327becb3c0e151b09654b0/68747470733a2f2f70726f6a656374732e776f6a74656b6d616a2e706c2f72656163742d63616c656e6461722f72656163742d63616c656e6461722e6a7067"><img width="436" height="398" src="https://camo.githubusercontent.com/52917c4e3af39a7ca8c3b140dcd4aab5f51aae98a8327becb3c0e151b09654b0/68747470733a2f2f70726f6a656374732e776f6a74656b6d616a2e706c2f72656163742d63616c656e6461722f72656163742d63616c656e6461722e6a7067" data-canonical-src="https://projects.wojtekmaj.pl/react-calendar/react-calendar.jpg" style="max-width: 100%;"></a>
</div>
<p>Ultimate calendar for your React app.</p>
<ul>
<li>Pick days, months, years, or even decades</li>
<li>Supports range selection</li>
<li>Supports virtually any language</li>
<li>No moment.js needed</li>
</ul>
<div class="markdown-heading"><h2 class="heading-element">tl;dr</h2><a id="user-content-tldr" class="anchor" aria-label="Permalink: tl;dr" href="#tldr"><span aria-hidden="true" class="octicon octicon-link"></span></a></div>
<ul>
<li>Install by executing <code>npm install react-calendar</code> or <code>yarn add react-calendar</code>.</li>
<li>Import by adding <code>import Calendar from 'react-calendar'</code>.</li>
<li>Use by adding <code>&lt;Calendar /&gt;</code>. Use <code>onChange</code> prop for getting new values.</li>
</ul>

<div class="markdown-heading"><h2 class="heading-element">Getting started</h2><a id="user-content-getting-started" class="anchor" aria-label="Permalink: Getting started" href="#getting-started"><span aria-hidden="true" class="octicon octicon-link"></span></a></div>
<div class="markdown-heading"><h3 class="heading-element">Compatibility</h3><a id="user-content-compatibility" class="anchor" aria-label="Permalink: Compatibility" href="#compatibility"><span aria-hidden="true" class="octicon octicon-link"></span></a></div>
<p>Your project needs to use React 16.8 or later.</p>
<p>react-calendar uses modern web technologies. That's why it's so fast, lightweight and easy to style. This, however, comes at a cost of <a href="https://caniuse.com/#feat=internationalization" rel="nofollow">supporting only modern browsers</a>.</p>
<div class="markdown-heading"><h4 class="heading-element">Legacy browsers</h4><a id="user-content-legacy-browsers" class="anchor" aria-label="Permalink: Legacy browsers" href="#legacy-browsers"><span aria-hidden="true" class="octicon octicon-link"></span></a></div>
<p>If you need to support legacy browsers like Internet Explorer 10, you will need to use <a href="https://github.com/andyearnshaw/Intl.js/">Intl.js</a> or another Intl polyfill along with react-calendar.</p>
<div class="markdown-heading"><h4 class="heading-element">My locale isn't supported!
